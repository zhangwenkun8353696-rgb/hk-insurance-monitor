# 香港保险舆论监测系统

HK Insurance Sentiment Monitor — 一站式香港保险行业舆论监测工具。

## 功能特色

- 📰 **舆情看板** — 每日舆情总览、情感分布、话题分类
- 🔴 **负面预警** — AI 情感分析自动标记负面新闻，分级风险预警
- 🏢 **公司关联** — 自动识别涉及的保险公司（友邦、保诚、安盛、宏利等10家）
- 💡 **应对策略** — 基于舆情类型自动生成公关应对建议
- 📊 **多维筛选** — 按情感、公司、话题、风险等级筛选

## 快速开始

```bash
# 安装依赖
pip install flask flask-cors

# 启动服务
python app.py

# 访问
http://127.0.0.1:5000
```

## 技术栈

- **后端**: Python Flask
- **前端**: HTML / CSS / JavaScript + Chart.js
- **分析**: 基于规则的情感分析、实体识别、话题分类
